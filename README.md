# Nuke Cog

Red-DiscordBot 서버 정리용 Cog 입니다. 특정 사용자만 실행 가능하며, 2단계 확인 후 채널/역할을 삭제합니다.

## 명령어

### [p]nuke
- 설명: 2단계 확인 후 서버 정리를 시작합니다.
- 동작:
  - 명령어 메시지 즉시 삭제
  - "fr?" 메시지 전송 후 ✅ 확인
  - 30초 내 확인 시 채널 → 역할 순서로 삭제
  - 진행 상황 및 완료/중단 결과를 DM으로 안내
- 조건:
  - 사용자 ID: `1173942304927645786`만 실행 가능
  - 봇 권한: Manage Channels, Manage Roles 필요
  - 진행 중 중복 실행 불가

### [p]nukestop
- 설명: 진행 중인 서버 정리를 즉시 중단합니다.
- 동작:
  - 삭제 루프 중단
  - 현재까지 삭제된 채널/역할 수를 DM으로 안내
- 조건:
  - 사용자 ID: `1173942304927645786`만 실행 가능
  - 진행 중일 때만 동작
